语义化标签

语义化标签，就是让标签有自己的含义，利用本身传达它所包含内容的一些信息，使浏览器和搜索引擎直观的认识标签和属性的用途和作用。有一下好处

  • 代码结构清晰，可读性高，减少差异化，便于团队开发和维护。
  • 在页面没有加载CSS的情况下，也能呈现良好的内容结构，提升用户体验。
  • 对搜索引擎友好，良好的结构和语义，有助于爬虫抓取更多的有效信息。
